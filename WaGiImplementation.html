Our application was made using HTML for pages structure, CSS for page stilization and JS for adding special functionalities and behaviors. In total, we have 3 pages, login (with signup) where you can join our community, home page where you can learn things about our application and the play page where you can actually test your knowledge and to collect points.

The login will be made using Stack Exchange account via Stack Exchange Login API.

The home page was made n scope to introduce users in the game idea, here, the user can find answers for questions like "What is WAGI?", "Why to use WAGI?" and "How you can use WAGI."

On play page you can start to interact with the game environment. Here you can see questions posted on pages like Stack Exchange and some posible responses, from those responses you have to chose the right one and to catch it if you can.
