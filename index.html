What Is FERT ?

FEedback in Real Time Is a "homemade" website designed to help public speakers interact with the audience.

It is entirely on the web with no need to download anything. Of course that means you need to have access to the internet.It's 2018 let's be real you can easily get wifi/ethernet access.

It works in both ways.What do we mean by that ?

Someone from the audience can ask a question in a session held by the speaker. The audience then decides if the question is relevant or could clear up a topic by upvoting it or downvoting. Same goes for the speaker. He can ask the audience a question to make sure everyone understood and that he can move forward with the speech.

It is open source. What that means ?

Well let's say it is free ! Even better , the source code is public on Github.

Why Use It ?

We designed this tool to help people who often cant reach their audience. This aspect sometimes is critical in delivering a successful presentation/speech.

Even as a audience you feel the need sometimes to reach out to the speaker , in order to fully understand the topic simply by adressing questions or asking for a clarification.

We tryed to make it simple to use and intuitive. Anyone can use it anywhere. With the code being open source you can make a institution only feedback website to use internally.

How You Can Use It ?

In order to use our tool you need to make a account and log in. We dont intend to use the data like Google we just want to know who asks the questions and who helds the sessions.

After you are authenticated you can go to the sessions page and search for the desired session. Anyone can hold a session and anyone can join so be nice !

From there you can see what questions others have asked , relevant information about the session and many other things.

Dont be afraid to use our website on mobile. We designed it so you can use it on any device !
